# Prompt Queue

Folder ini digunakan oleh anggota tim yang tidak punya akses langsung ke Claude untuk menaruh permintaan koordinasi.

## Cara Pakai

1. Buat file baru di folder ini dengan format nama: `YYYY-MM-DD_deskripsi-singkat.md`
2. Isi file dengan permintaan koordinasi (misal: minta breakdown task baru, update status, dll.)
3. Claude Cowork akan memproses isi file ini dan menaruh hasilnya di `03-output-review/`

## Contoh Format File

```
# Permintaan: [judul singkat]
**Dari:** [nama/peran]
**Tanggal:** YYYY-MM-DD

## Isi Permintaan
[tulis permintaan di sini]
```
